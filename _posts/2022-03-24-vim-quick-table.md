## Short Cut Key

 按键 | 功能 | 备注
---------|----------|---------
 Esc | normal | 正常模式
 i | insert | 插入模式
 : | command | 命令行模式
 h | | 左移一个字符
 j | | 下移一个字符
 l | | 右移一个字符
 k | | 上移一个字符
 0 | | 行首
 $ | | 行尾
